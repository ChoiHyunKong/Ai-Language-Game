# 점수 및 랭킹 시스템 지침서

점수 계산, 콤보 시스템, 로컬/글로벌 랭킹 구현 가이드

---

## 1. 점수 시스템 구성
- 산출 공식: 총 점수 = 기본 점수 + 속도 보너스 + 콤보 보너스

---

## 2. 점수 계산 공식

### 2.1 기본 점수
- 난이도 1 (쉬움): 10점 (3글자 이하)
- 난이도 2 (보통): 20점 (4-5글자)
- 난이도 3 (중간): 35점 (6-7글자)
- 난이도 4 (어려움): 50점 (8-9글자)
- 난이도 5 (매우 어려움): 70점 (10글자 이상)

### 2.2 속도 보너스
- 기준: 단어 출현 후 입력 완료 시점까지 남은 시간 비중
- 보너스 구간: 매우 빠름(50점), 빠름(30점), 보통(15점), 느림(5점)

### 2.3 콤보 보너스
- 적용 방식: 연속 정답 시 배수 적용
- 임계치별 배수: 5콤보(1.2x), 10콤보(1.5x), 20콤보(2.0x), 30콤보(2.5x), 50콤보(3.0x)

---

## 3. 콤보 시스템 규칙
- 상승: 정답 입력 시 콤보 1 증가
- 유지: 오답 입력 시 현재 콤보 유지
- 초기화: 단어가 바닥에 도달하여 실점할 경우 0으로 리셋
- 효과: 특정 콤보 달성 시 NICE, GREAT, AMAZING 등의 시각 효과 및 사운드 출력

---

## 4. 로컬 랭킹 시스템
- 기술: LocalStorage 활용
- 데이터 관리: 최대 100개 기록 저장, 점수순 정렬, 개인 최고 기록 추적
- 필드 구성: id, playerName, score, combo, mode, difficulty, date

---

## 5. 글로벌 랭킹 시스템
- 기술: Firebase Realtime Database 연동
- 운영 방식: 전 세계 사용자 점수 통합 관리, 상위 100위 조회
- 보안: Firebase Security Rules를 통한 데이터 무결성 보호

---

## 6. 결과 화면 구성
- 통계 항목: 최종 점수, 최대 콤보, 정확도, 도달 레벨 표시
- 기능: 사용자 이름 입력, 점수 제출, 다시 시작, 메뉴 이동

---

## 7. 구현 체크리스트
- 점수 계산 로직 및 콤보 시스템 개발
- 콤보 마일스톤별 시각 효과 구현
- 로컬 저장소 기반 랭킹 관리 클래스 구축
- Firebase 실시간 데이터베이스 연동 및 랭킹 보드 개발
- 게임 종료 후 결과 요약 및 데이터 제출 인터페이스 구현
