# 📌 프로젝트 개요

> **AI 기반 단어 퀴즈 타자 게임**  
> 한컴 타자 연습 '산성비' 게임을 벤치마킹한 차세대 학습용 타자 게임

---

## 1. 프로젝트 비전

### 목표
1. **기존 한컴 타자의 한계 극복**
   - 단순 타이핑을 넘어 단어의 의미, 예문, 활용법 학습
   
2. **운영 효율 극대화**
   - 서버 비용 0원 (무료 호스팅)
   - CSV 기반 간편 콘텐츠 업데이트
   - 영구적 가용성 확보

3. **사용자 경험**
   - AI를 통한 실시간 퀴즈 생성
   - 다국어 확장성 제공

## 3. 핵심 기능

### 3.1 게임 모드

| 모드 | 설명 | 입력 |
|------|------|------|
| 🔤 단어 연습 | 떨어지는 단어를 보고 입력 | 단어 그대로 |
| 🧠 의미 퀴즈 | 뜻을 보고 단어 유추 | 해당 단어 |
| ✏️ 문장 완성 | 예문의 빈칸 채우기 | 빈칸에 들어갈 단어 |

### 3.2 AI 기능 (WebLLM)
- 유사 단어 실시간 생성
- 힌트 제공
- 난이도 자동 조절

### 3.3 점수 시스템
- 난이도별 차등 점수
- 속도 보너스
- 콤보 시스템

---

## 4. 기술 스택

| 분류 | 기술 | 용도 |
|------|------|------|
| **Frontend** | HTML5, CSS3, Vanilla JS | UI 및 게임 로직 |
| **AI** | WebLLM | 브라우저 로컬 AI 엔진 |
| **Data** | CSV (PapaParse) | 퀴즈 데이터 관리 |
| **Storage** | LocalStorage | 개인 기록 저장 |
| **Ranking** | Google Sheets API | 글로벌 랭킹 |
| **Hosting** | Vercel | 무료 정적 호스팅 |
| **Version Control** | GitHub | 형상 관리 |
