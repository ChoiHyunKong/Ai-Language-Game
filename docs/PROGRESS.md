# 개발 진행 리포트

본 문서는 AI 지식 게임의 개발 이력 및 향후 계획을 통합 관리합니다.

---

## 1. 프로젝트 요약
- 목표: AI 관련 전문 용어 기반의 학습용 타자 연습 게임 개발
- 핵심 구성: Vanilla JS 엔진, CSV 데이터 로더, Firebase 실시간 랭킹 시스템

---

## 2. 주요 구현 내역

### 2.1 코어 프레임워크
- 게임 엔진: Canvas 기반 낙하 로직 및 충돌 감지 처리
- 데이터 로더: CSV 기반 다국어 파싱 및 동적 로드 체계 구축
- 화면 전환: ScreenManager를 통한 UI 단계별 상태 제어

### 2.2 게임 메커니즘
- 지표 시스템: 점수, 콤보, 속도 보너스, 정확도 산출 로직 구현
- 특수 요소: 골든 워드(점수 배수), 동적 난이도 상승 시스템 적용
- 사용자 피드백: 입력 히스토리 패널 및 단어 의미 모달 팝업 제공

### 2.3 시각 및 음향 효과
- 디자인 테마: 네온 사이버펑크 스타일의 CSS 시스템 구축
- 배경 연출: 그라데이션, 그리드 애니메이션, 파티클 레이어 적용
- 오디오: Web Audio API 기반의 상황별 효과음 8종 탑재

### 2.4 인프라 및 보안
- 랭킹 시스템: Firebase 기반 글로벌 순위 공유 및 데이터 무결성 검증 추가
- 프로젝트 환경: Vite 빌드 환경 전환 및 소듈화 구조 개선

---

## 3. 향후 로드맵

### 1단계: 안정화 및 확장
- PWA 최적화: 오프라인 캐싱 강화 및 설치 아이콘 품질 개선
- 다국어 추가: 일본어 외 신규 지원 언어 데이터 확보

### 2단계: 유입 극대화
- SEO 강화: 메타 태그 최적화 및 검색 로봇 등록
- 마케팅: IT 커뮤니티 및 SNS 플랫폼 유입 채널 구축

### 3단계: 신규 콘텐츠
- 의미 퀴즈 모드: 뜻 기반 유추 게임성 강화
- 문장 완성 모드: AI 맥락 데이터 처리 고도화
