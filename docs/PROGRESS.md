# AI 지식 게임 (AI Word Quiz) - 개발 진행 리포트

본 문서는 AI 지식 게임의 개발 진행 상황과 구현된 기능, 수정된 버그 및 향후 계획을 기록합니다.

## 📋 프로젝트 개요
- **목표**: 사용자가 타자 연습을 통해 AI 관련 전문 용어와 지식을 자연스럽게 습득할 수 있는 웹 기반 게임.
- **영감**: 한컴 타자연습 '산성비' 모드.
- **기술 스택**: Vanilla JavaScript, HTML5 Canvas, CSS3, Web Audio API, PWA.

---

## ✅ 현재까지의 진행 사항

### 1. 핵심 프레임워크 구축
- [x] 프로젝트 구조 설정 (Public, Data, Docs 폴더 분리)
- [x] **GameEngine.js**: Canvas 기반의 워드 렌더링 및 물리 엔진 구현
- [x] **CSVLoader.js**: `words.csv` 파일을 통한 다국어(한/영/일) 단어 데이터 관리
- [x] **LanguageManager.js**: 게임 내 실시간 언어 전환 시스템
- [x] **ScoreManager.js**: LocalStorage 기반의 랭킹 및 최고 기록 관리
- [x] **ScreenManager.js**: 메뉴, 게임, 결과 화면 간의 매끄러운 전환

### 2. 게임 기능 및 메커니즘 고도화
- [x] **중복 단어 방지**: 한 게임 세션 내에서 이미 입력 성공한 단어는 나타나지 않음.
- [x] **골든 워드 (Golden Word)**: 15% 확률로 특수 단어 등장, 입력 시 점수 2배 획득.
- [x] **동적 난이도 조절**: 단어 5개 입력 시마다 낙하 속도 30% 증가.
- [x] **입력 히스토리 패널**: 우측 패널에 성공한 단어들이 아래에서부터 쌓이며 기록됨.
- [x] **단어 학습 기능**: 기록된 단어 블록을 클릭하면 해당 단어의 의미와 예문을 보여주는 모달 팝업 표시.
- [x] **오타 페널티**: 잘못된 입력 시 라이프 감소 및 콤보 초기화.

### 3. UI/UX 및 시각 효과 개선
- [x] **네온 사이버펑크 테마**: 다크 모드 기반의 화려한 네온 컬러 시스템 적용.
- [x] **애니메이션 배경**: 3중 레이어(그라디언트, 움직이는 그리드, 부유 파티클) 배경 구현.
- [x] **게임 종료 연출**: 게임 오버 시 즉시 결과를 보여주는 대신, 화면을 유지하여 마지막 상태를 복기하고 단어 학습을 할 수 있도록 오버레이 시스템 구축.
- [x] **반응형 디자인**: 다양한 화면 크기에 대응하는 레이아웃 최적화.

### 4. 오디오 시스템
- [x] **SoundManager.js**: Web Audio API를 사용하여 외부 파일 없이 효과음 생성.
- [x] **다양한 효과음**: 정답, 오답, 콤보(상승음), 골든워드(반짝임), 게임오버, 속도증가 등 8종 이상의 효과음 적용.
- [x] **음소거 기능**: 전역 음소거 버튼 및 UI 상태 표시.

### 5. 버그 수정 및 최적화
- [x] **브라우저 호환성**: `roundRect` 미지원 브라우저용 폴리필(Polyfill) 구현.
- [x] **초기 로딩 최적화**: 캔버스 크기 계산 오류로 인한 프로그램 종료 문제 해결.
- [x] **클릭 방해 문제**: 애니메이션 배경이 버튼 클릭을 막는 현상 해결 (`z-index` 및 `pointer-events` 속성 조정).
